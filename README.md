# AI website
website designed with the use of gemini artificial intelligence to demonstrate how to work with AI.  
The personal website I created is about me. It has my resume, skills, social media links, and an area for projects and organizations I am a part of.  
The following features are implemented in the project:  
reactionary buttons  
persistent dropdown menu  
static background image  
important sections of the website react to the user moving their mouse over them

As I was working on the website most of the references I looked at were very minimalist which gave me the impression that I should also design my personal website like that.  
At some point in the creative process I discovered a website that featured different css styles that I could use to give my website life without being overbearing or hard to navigate with visual bloat.  
The way it is now all of the creative ideas were in making navigating the website and interacting with it a more lively process without detracting from its main goal of delivering information to the user.  
If I were to keep working on the website I would likely add more detail to formatting and page layout fitting into a guideline or ruleset that webpage designers likely use.
